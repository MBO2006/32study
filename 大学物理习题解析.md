# 第一部分：机械波

## <span id="机械波判断题">一、判断题</span>

**1.** 在波传播方向上相邻的两个位移相同点的距离就是波长。（ ）
**答：×**。波长是相邻的两个**相位相同**的点（同相点）之间的距离。位移相同不一定相位相同。

**2.** 我们经常会看到"长江后浪推前浪"的现象，这说明质点随波一起移动。（ ）
**答：×**。波传播的是振动形式和能量，**质点并不随波迁移**，只是在平衡位置附近振动。

**3.** 一平面简谐波传播时，在传播方向上，某媒质质元在负的最大位移处，则它的动能最大，势能最大。（ ）
**答：×**。质元在最大位移处时速度为0，动能为0；形变为0，势能也为0。

## <span id="机械波单选题">二、单选题</span>

**1.** 频率为100Hz，传播速度为300m/s的平面简谐波，波线上两点的相位差为π/3，则此两点间相距为（ ）

A. 2m  
B. 2.19m  
**C. 0.5m**  
D. 28.6m

**解析：**
波长 λ = v/f = 300/100 = 3m  
相位差 Δφ = 2π·Δx/λ  
Δx = λ·Δφ/(2π) = 3 × (π/3)/(2π) = **0.5m**
- **答案：C**

---

**2.** 一平面简谐波在弹性媒质中传播时，在传播方向上，某媒质质元在负的最大位移处，则它的能量为（ ）

A. 动能为零，势能最大  
**B. 动能为零，势能为零**  
C. 动能最大，势能最大  
D. 动能最大，势能为零

**解析：**
在最大位移处，质元速度 v=0，动能为0；同时相对形变量为0（速度为零处形变也为零），所以弹性势能为0。波动中质元的动能和势能**同相位**变化，这点与简谐振动不同。
- **答案：B**

---

**3.** 一平面简谐波在弹性媒质中传播，在媒质质元从平衡位置运动到最大位移处的过程中（ ）

A. 它的动能转换成势能  
B. 它的势能转换成动能  
C. 它从相邻的一段质元获得能量其能量逐渐增大  
**D. 它把自己的能量传给相邻的一段质元，其能量逐渐减小**

**解析：**
质元从平衡位置向最大位移运动时，速度减小、动能减小，同时形变减小、势能也减小（波动中动能势能同相），总机械能减小。它把能量传递给相邻质元。
- **答案：D**

---

**4.** 波从一种介质进入另一种介质时，以下不变的量是（ ）

A. 波长  
**B. 频率**  
C. 波速  
D. 振幅

**解析：**
波的频率由波源决定，与介质无关。波速由介质决定（v = √(弹性/惯性)），波长 λ = v/f 也随介质改变。
- **答案：B**

---

**5.** 机械波的表达式为 y = 0.05cos(6πt + 0.06πx)（SI），则（ ）

A. 波长为5m  
B. 波速为10m/s  
**C. 周期为1/3 s**  
D. 波沿x轴正方向传播

**解析：**
对比标准形式 y = Acos(ωt ± kx + φ₀)：
- ω = 6π rad/s，T = 2π/ω = 2π/(6π) = **1/3s** ✓
- k = 0.06π rad/m，λ = 2π/k = 2π/(0.06π) ≈ 33.3m，A❌
- v = ω/k = 6π/(0.06π) = 100m/s，B❌
- 式中为"+"号，波沿x轴**负**方向传播，D❌
- **答案：C**

**6.** 一驻波在某时刻的波形曲线如图所示，在图中a、b、c三处质元振动的相位关系为（ ）

A. a与b相位相同，b与c相位相同  
B. a与b相位相同，b与c相位相反  
**C. a与b相位相反，b与c相位相同**  
D. a与b相位相反，b与c相位相反

**解析：**
驻波中，相邻波节之间的各点振动相位相同，波节两侧的点振动相位相反。从波形图看，a和b在波节两侧（相位相反），b和c在同一波节区间内（相位相同）。
- **答案：C**

**7.** 一平面简谐波沿x轴正向传播，在波线上有两质点，分别位于x₁和x₂，且x₂ > x₁，x₂ - x₁ < λ；x₁处质点和x₂处质点的振动曲线分别由曲线I和曲线II表示，则x₂处质点比x₁处质点振动相位落后（ ）

A. π/4  
**B. π/2**  
C. 3π/4  
D. 3π/2

**解析：**
沿波传播方向（x正向），下游质点的相位落后于上游质点。由振动曲线图可判断曲线II比曲线I落后π/2的相位。
- **答案：B**

**8.** 一平面简谐波沿x轴传播，已知在x₁=2m处质元的振动方程为 y₁ = 2cos(8πt - π/2)(SI)；在x₂=3m处质元的振动方程为 y₂ = 2cos(8πt + π/2)(SI)；x₂ - x₁ < λ，则该平面波的波长为（ ）

A. 0.5m  
B. 1m  
C. 1.5m  
**D. 2m**

**解析：**
从振动方程的常数项可知两处质点的相位差：
Δφ = π/2 - (-π/2) = π  
波程差 Δx = x₂ - x₁ = 1m  
Δφ = 2π·Δx/λ → π = 2π·1/λ → λ = **2m**
- **答案：D**

**9.** 一平面简谐波沿x轴正向传播，周期为T，t时刻的波形如图所示，由图可知，x₂处质元比x₁处质元开始振动的时间和相位分别落后（ ）

A. T/4 和 π/2  
B. T/4 和 3π/2  
C. 3T/4 和 π/2  
D. 3T/4 和 3π/2

**解析：**
从波形图判断，x₂在x₁的下游方向（波正向传播），x₂处质元开始振动的时间比x₁晚。时间落后 Δt = Δx/v，相位落后 Δφ = ωΔt = 2πΔt/T。根据波形图中x₁和x₂的相对位置及波长关系计算。
- **答案：需结合波形图具体判断**

**10.** 平面简谐波的表达式为 y = 0.02cos[π(20t - 5x) + π/2](SI)，其频率和波速分别为（ ）

**A. 10Hz 和 4m/s**  
B. 10Hz 和 5m/s  
C. 20Hz 和 4m/s  
D. 20Hz 和 5m/s

**解析：**
展开：y = 0.02cos(20πt - 5πx + π/2)  
对比标准形式：ω = 20π rad/s → f = ω/(2π) = **10Hz**  
k = 5π rad/m → v = ω/k = 20π/(5π) = **4m/s**
- **答案：A**

**11.** 下列函数y(x,t)可表示弹性介质中的波，式中A、a和b是正的常量。表示沿着x轴负方向传播的平面简谐波的表达式是（ ）

**A. y(x,t) = Acos(at + bx)**  
B. y(x,t) = Acos(at - bx)  
C. y(x,t) = Acos(at)·cos(bx)  
D. y(x,t) = Asin(at)·sin(bx)

**解析：**
- 正方向传播：y = Acos(ωt - kx)  
- 负方向传播：y = Acos(ωt + kx) → **A✓**  
- C和D为驻波形式（变量分离）
- **答案：A**

**12.** 下列叙述中正确的是（ ）

**A. 频率不同的光一定不能相互干涉**  
B. 频率不同的光一定能够相互干涉  
C. 频率相同的光一定不能相互干涉  
D. 频率相同的光一定能够相互干涉

**解析：**
干涉条件：频率相同、振动方向相同、相位差恒定。A✓。但频率相同的光不一定能干涉（还需满足其它两个条件），所以D❌。
- **答案：A**

**13.** 频率为500Hz的机械波，波速为360m/s，则同一波线上相位差为π/3的两点间距离为（ ）

A. 0.48m  
B. 0.36m  
C. 0.24m  
**D. 0.12m**

**解析：**
λ = v/f = 360/500 = 0.72m  
Δx = λ·Δφ/(2π) = 0.72 × (π/3)/(2π) = **0.12m**
- **答案：D**

**14.** 平面简谐波方程为 y = Acos(Bt - Cx)(m)，A、B、C为正恒量，则（ ）

A. 波速为 B/C  
B. 周期为 2π/B  
C. 波长为 2π/C  
D. 圆频率为 B

**解析：**
对照 y = Acos(ωt - kx)：ω = B，k = C  
波速 v = ω/k = **B/C** ✓  
周期 T = 2π/ω = **2π/B** ✓  
波长 λ = 2π/k = **2π/C** ✓  
圆频率 ω = **B** ✓
- **答案：A、B、C、D（全对）**

**15.** 在下列关于机械波的能量的表述中，正确的是（ ）

A. 波中一质元的能量 E = Ek + Ep = ½ρω²A²，其中A为波幅  
B. 机械波在介质中传播时，任一质元的Ek和Ep均随时间t变化，但相位相差π/2  
**C. 波中质元的Ek和Ep同相位，表明波的传播是能量传递的过程**  
D. 由于Ek和Ep同时为零，又同时达到最大值，表明质元运动不是振动

**解析：**
A❌：质元能量 E = ½ρω²A²ΔV（体元），不是点。  
B❌：波动中Ek和Ep**同相位**（同时最大、同时为零）。  
C✓：正确。  
D❌：这正是受迫振动的表现，质元仍然在振动。
- **答案：C**

**16.** 关于波下述说法中正确的是（ ）

A. 波只能分为横波和纵波  
B. 波动质点按波速向前运动  
**C. 波动中传播的只是运动状态和能量**  
D. 波经过不同媒质传播过程中波长不变

**解析：**
A❌：还有表面波等。  
B❌：质点只在平衡位置附近振动，不随波迁移。  
C✓：波的本质是振动形式和能量的传播。  
D❌：介质改变 → 波速改变 → 波长改变（频率不变）。
- **答案：C**

**17.** 对于机械横波，下列说法正确的是（ ）

**A. 波峰处质元的动能、势能均为零**  
B. 处于平衡位置的质元势能为零，动能为最大  
C. 处于平衡位置的质元动能为零，势能为最大  
D. 波谷处质元的动能为零，势能为最大

**解析：**
波动中动能势能同相位：  
波峰处：速度=0 → 动能=0；形变=0 → 势能=0 → **A✓**  
平衡位置：速度最大 → 动能最大；形变最大 → 势能最大 → B、C❌  
波谷处：与波峰处同理，动能势能均为零 → D❌
- **答案：A**

**18.** 若一平面简谐波的波动方程为 y = 0.05cos(6πt - 0.06πx)(m)，则（ ）

**B. 周期为1/3s**  
C. 波长为100m

# 第二部分：静电场

## <span id="静电场判断题">一、判断题</span>

**1.** 根据点电荷场强公式，当被场点距源点电荷很近(r→0)时，则场强E→∞。（ ）
**答：×**。点电荷模型在r→0时不再适用，此时电荷不能视为点电荷。

**2.** 沿着电场线移动负电荷，负电荷的电势能是减少的。（ ）
**答：×**。电场线指向电势降低的方向。负电荷沿电场线移动时，电势降低，但负电荷的电势能 W = qU（q<0），U减小 → W**增大**。

**3.** 在 E = F/q 和 E = kQ/r² 两式中，E的含义相同。（ ）
**答：×**。E = F/q 是场强的**定义式**（普通适用），E = kQ/r² 是**点电荷的场强公式**（仅适用于点电荷）。

**4.** 接地空腔导体只能屏蔽空腔内电场。（ ）
**答：×**。接地的空腔导体**既能屏蔽内部电场对外部的影响，又能屏蔽外部电场对内部的影响**。

**5.** 由静电场高斯定理，闭合曲面电通量为零时，曲面上电场强度为零。（ ）
**答：×**。电通量为零只说明闭合曲面内电荷代数和为零，曲面上各点场强不一定为零。

**6.** 处于静电平衡状态的导体，内部场强处处为零的原因是所有感应电荷在导体内部产生的合场强为零。（ ）
**答：×**。是**外电场和所有感应电荷的电场在导体内部叠加的结果为零**，而非仅感应电荷。

**7.** 点电荷放在球形高斯面的中心处，将球心处的点电荷移到高斯面内另一处，高斯面的电通量发生变化。（ ）
**答：×**。高斯面内电荷量不变，电通量就不变（与位置无关）。

**8.** 空腔导体可以同时屏蔽空腔内、外电场。（ ）
**答：√**。空腔导体（需接地）可同时屏蔽内外电场。

**9.** 由点电荷场强公式，场点距源点电荷很近(r→0)时，场强E→∞。（ ）
**答：×**。同第1题，点电荷模型在极近距离不适用。

**10.** 当一个带电导体达到静电平衡时表面曲率较大处电势较高。（ ）
**答：×**。静电平衡时导体是等势体，**表面各处电势相等**。曲率大处电荷面密度大、场强大，但电势处处相等。

## <span id="静电场单选题">二、单选题</span>

**1.** 在一个原来不带电的外表面为球形的空腔导体A内，放一带电荷为+Q的带电导体B，如图所示。比较空腔导体A的电势UA和导体B的电势UB时，可得以下结论（ ）

A. UA = UB  
B. UA > UB  
**C. UA < UB**  
D. 因空腔形状不是球形，两者无法比较

**解析：**
B带正电，A内表面感应负电荷，外表面感应正电荷。由于B带正电且被A包围，电场线从B指向A内表面，沿电场线方向电势降低，所以 **UA < UB**。
- **答案：C**

**2.** 如图所示，直线MN长为2R，弧NPM是以O为中心、R为半径的半圆弧，O点有正电荷+q，M点有负电荷-q。今将一试验电荷+q₀从P点沿路径PNM移到无穷远处，设无穷远处电势为零，则电场力作功（ ）

A. W < 0，且为有限常量  
B. W > 0，且为有限常量  
C. W → ∞  
**D. W = 0**

**解析：**
O点的+q和M点的-q构成电偶极子。P点位于以O为圆心的半圆弧上，且O处+q到P的距离=R，M处-q到P的距离=√((2R)²+R²)=√5R。
实际上，由于O(-q)和M(+q)？不对，O点有+q，M点有-q。
P点的电势由两者叠加。但题目中q₀从P沿路径到无穷远，电场力做功 W = q₀(U_P - U_∞) = q₀U_P。
分析P点的电势：UP = kq/R + k(-q)/MP = kq/R - kq/MP
但这不是电偶极子的中垂线。需要具体看P的位置。
实际上，如果O和M的电荷等值异号，且P在半圆弧上，考虑到PNM路径，可能由于对称性使电场力做功为0。
- **答案：D**

**3.** 真空中两块互相平行的无限大均匀带电平面，其电荷密度分别为+σ和+2σ，两板之间的距离为d，两板间的电场强度大小为（ ）

A. 0  
B. 3σ/(2ε₀)  
C. σ/ε₀  
**D. σ/(2ε₀)**

**解析：**
无限大均匀带电平面产生匀强电场 E = σ/(2ε₀)，方向背离板面。
第一块板（+σ）：E₁ = σ/(2ε₀)，向右  
第二块板（+2σ）：E₂ = 2σ/(2ε₀) = σ/ε₀，向左（在两板间）  
两板间合场强：E = E₂ - E₁ = σ/ε₀ - σ/(2ε₀) = **σ/(2ε₀)**
- **答案：D**

**4.** 已知一高斯面所包围的体积内电量代数和为零，则可肯定（ ）

A. 高斯面上各点场强均为零  
B. 穿过高斯面上每一面元的电通量均为零  
**C. 穿过整个高斯面的电通量为零**  
D. 以上说法都不对

**解析：**
由高斯定理 ∮E·dS = Σq/ε₀，Σq = 0 → 总通量为零 ✓。但面上各点场强不一定为零，面元电通量也不一定为零。

**26.** 二平行无限大均匀带电平板的面电荷密度分别为σ₁和σ₂，则二板间电场强度的大小为（ ）

A. |σ₁ - σ₂|/(2ε₀)  
B. (σ₁ + σ₂)/(2ε₀)  
C. |σ₁ - σ₂|/ε₀  
D. (σ₁ + σ₂)/ε₀

**解析：**
两无限大带电平面在板间产生的场强方向取决于电荷符号。若σ₁=σ（正），σ₂=-σ（负）：板间 E = σ/ε₀ + σ/ε₀... 不对。

一般情况：板间 E = |σ₁ - σ₂|/(2ε₀)（同号）或 (|σ₁|+|σ₂|)/(2ε₀)（异号）。
- **答案：A**

**1.** 说明导体处于静电平衡状态时，导体的场强和电势特征，如为实心导体，电荷分布的特征。

**答：**
- **场强特征**：导体内部场强处处为零；导体表面外侧场强方向与表面垂直，大小 E = σ/ε₀。

# 第三部分：光的干涉

## <span id="光的干涉判断题">一、判断题</span>

**1.** 在劳埃德镜干涉装置中，平面镜与屏交界处的干涉条纹应为暗条纹。（ ）
**答：√**。劳埃德镜实验中，反射光有半波损失，在镜与屏交界处光程差为λ/2，形成暗纹。

**2.** 在杨氏实验中，相邻亮条纹中心或相邻两暗条纹中心的距离一定相等。（ ）
**答：√**。杨氏双缝干涉条纹是等间距的，Δx = λD/d，与干涉级次无关。

**3.** 用单色光垂直照射杨氏双缝，减小双缝间距，干涉条纹间距变小。（ ）
**答：×**。条纹间距 Δx = λD/d，减小d（缝间距）→ Δx**增大**。

**4.** 在杨氏双缝干涉实验中，若用薄玻璃片盖住上缝，干涉条纹将下移。（ ）
**答：×**。盖住上缝后上缝光程增加 δ=(n-1)t。要补偿此额外光程差，中央零级处上缝几何路程需**小于**下缝几何路程 ⇒ 条纹朝**上缝方向移动 ⇒ 上移**。

**解析：**

# 第四部分：磁场与电磁感应

## <span id="磁场判断题">一、判断题</span>

**1.** 在尺寸相同的铁环和铜环所包围的面积中穿过相同变化率的磁通量，则两环中，感应电动势不同，感应电流不同。（ ）
**答：×**。感应电动势 ε = -dΦ/dt 相同（与材料无关），但感应电流 I = ε/R 不同（铜电阻率小，感应电流大）。

**2.** 感生电场是由变化的磁场激发的，它的电场线是不闭合的。（ ）
**答：×**。感生电场的电场线是**闭合的**（涡旋电场），这是与静电场的本质区别之一。

**3.** 由法拉第电磁感应定律知（设回路的总电阻一定），穿过闭合电路的磁通量达最大时，回路中的感应电流达最大。（ ）
**答：×**。感应电流与磁通量的**变化率**成正比（I = ε/R = (dΦ/dt)/R），而非与磁通量本身成正比。磁通量最大时变化率通常为零，此时感应电流为零。

**4.** 磁感应强度沿闭合回路积分为零时，回路中穿过的电流代数和为零。（ ）
**答：√**。由安培环路定理 ∮B·dl = μ₀ΣI，积分为零 ⇒ ΣI = 0。

**5.** 感生电场是由电荷激发，它的电场线是不闭合的。（ ）
**答：×**。感生电场由**变化的磁场**激发（而非电荷），它的电场线是**闭合的**。

## <span id="磁场简答题">三、简答题</span>

**1.** 写出静电场和稳恒磁场的高斯定律和环路定律的数学表达式以及总结静电场和稳恒磁场的不通属性。

**答：**

| 定律 | 静电场 | 稳恒磁场 |
|------|--------|----------|
| 高斯定律 | ∮E·dS = Σq/ε₀ | ∮B·dS = 0 |
| 环路定律 | ∮E·dl = 0 | ∮B·dl = μ₀ΣI |

**不同属性：**
- **静电场**：**有源无旋场**——电场线发自正电荷、止于负电荷（不闭合），电场力做功与路径无关
- **稳恒磁场**：**无源有旋场**——磁感线闭合（无起点终点），不存在磁单极子

**2.** 写出静电场和感生电场的相同点和不同点。

**答：**

| 比较项 | 静电场 | 感生电场 |
|--------|--------|----------|
| **激发方式** | 由**静止电荷**激发 | 由**变化的磁场**激发 |
| **电场线** | **不闭合**（发自正电荷止于负电荷） | **闭合**（涡旋状） |
| **环路定理** | ∮E·dl = **0**（保守场、无旋场） | ∮E·dl = **-∫∂B/∂t·dS**（非保守） |
| **高斯定理** | ∮E·dS = **Σq/ε₀**（有源场） | ∮E·dS = **0**（无源场） |

**相同点：** 都对电荷有力的作用（F=qE），都用电场强度E描述，都满足叠加原理。

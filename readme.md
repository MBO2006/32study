## 3. 系统总体设计

### 3.1 系统框图

```
                        ┌────────────────────────────────────────────┐
                        │             数控直流电源系统                  │
                        │                                            │
   ┌──────────┐         │  ┌──────────┐    ┌────────────┐            │
   │ 28~32V   │─────────┼─▶│ 输入滤波  │───▶│  同步Buck   │──▶ 输出    │
   │ 直流输入  │         │  │ C5,C7    │    │ IR2104+2×  │    5~24V   │
   │ P1       │         │  │ 1000μF   │    │ IRF540+1mH │    0~3A    │
   └──────────┘         │  └──────────┘    └─────┬──────┘            │
                        │                       │                    │
   ┌──────────┐         │  ┌──────────┐    ┌─────┴──────┐            │
   │ +12V辅助  │─────────┼─▶│ 辅助电源  │    │  输出滤波   │            │
   │ 电源模块  │         │  │ 给IC供电  │    │ C6,C8      │            │
   │ (商用)    │         │  └────┬─────┘    │ 1000μF     │            │
   └──────────┘         │       │          └────────────┘            │
                        │       │                                    │
                        │  ┌────┴─────────────────────────────┐      │
                        │  │           MCU 控制核心             │      │
                        │  │        STM32F103C8T6              │      │
                        │  │                                   │      │
                        │  │  PWM ──────────▶ IR2104 IN        │      │
                        │  │  ADC0 ◀── 输出电压采样 (分压)       │      │
                        │  │  ADC1 ◀── 输出电流采样 (运放)       │      │
                        │  │  GPIO ───────▶ SD# 使能/关断      │      │
                        │  │  I2C ────────▶ OLED 显示屏        │      │
                        │  │  GPIO ◀────── 按键 / 编码器       │      │
                        │  │  GPIO ───────▶ 蜂鸣器/指示灯       │      │
                        │  └──────────────────────────────────┘      │
                        └────────────────────────────────────────────┘
```

**工作原理简述：**

1. 直流 28~32V 输入经过滤波电容进入 Buck 功率级
2. MCU 输出 PWM 信号给 IR2104 半桥驱动器
3. IR2104 驱动两个 IRF540 MOSFET，将高压 PWM 斩波
4. 经 1mH 电感 + 1000μF 电容滤波后输出平滑直流
5. MCU 通过 ADC 采集输出电压/电流，运行 PID 算法调节 PWM 占空比，实现稳压
6. 用户通过按键/编码器设定目标电压，OLED 实时显示

**IR2104 引脚连接速查：**

| 引脚 | 名称 | 连接目标 | 说明 |
|:---:|------|----------|------|
| 1 | **VCC** | +12V（经 100Ω R1）→ C1(47μF)→AGND | 芯片供电 |
| 2 | **IN** | MCU PWM（PA6，3.3V 电平） | PWM 信号输入 |
| 3 | **SD#** | MCU GPIO（PB0），高=使能，低=关断 | 关断保护 |
| 4 | **COM** | AGND（信号地） | 芯片参考地 |
| 5 | **LO** | 经 R4(10Ω)→Q2 栅极 | 低侧 MOSFET 驱动 |
| 6 | **VS** | Q1 源极 = Q2 漏极 = L1 输入端 | 开关节点 |
| 7 | **HO** | 经 R2(10Ω)→Q1 栅极 | 高侧 MOSFET 驱动 |
| 8 | **VB** | D1 FR207→+12V，C2(47μF)→VS | 自举浮地供电 |

## 6. 关键设计要点

| 类别 | 要点 | 详细说明 |
|------|------|----------|
| 🔌 **供电** | 自举电路 | C2(47μF) 紧靠 IR2104 的 VB(8) 和 VS(6) 引脚 |

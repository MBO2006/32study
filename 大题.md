### 6.3.6 平面简谐波的描述

**【题目】** 已知波源在原点的一列平面简谐波，波函数为 $y = A\cos(Bt - Cx)$，其中 $A$，$B$，$C$ 为正值恒量．求：

1. 波的振幅、波速、频率、周期与波长；
2. 写出传播方向上距离波源为 $l$ 处一点的振动方程；
3. 任一时刻，在波的传播方向上相距为 $d$ 的两点的相位差．

**【解析】**

**(1)** 将已知波函数与平面简谐波的标准形式比较：

标准形式：$y = A\cos\left(2\pi\nu t - \frac{2\pi}{\lambda}x\right)$

已知：$y = A\cos(Bt - Cx)$

对比可得：
- **振幅**：$A$
- **角频率**：$\omega = B$，故**频率** $\nu = \dfrac{\omega}{2\pi} = \dfrac{B}{2\pi}$
- **周期**：$T = \dfrac{1}{\nu} = \dfrac{2\pi}{B}$
- **波数**：$k = C = \dfrac{2\pi}{\lambda}$，故**波长** $\lambda = \dfrac{2\pi}{C}$
- **波速**：$u = \lambda\nu = \dfrac{2\pi}{C} \cdot \dfrac{B}{2\pi} = \dfrac{B}{C}$

**(2)** 将 $x = l$ 代入波函数，可得距离波源 $l$ 处质点的振动方程：

$$y = A\cos(Bt - Cl)$$

**(3)** 同一时刻 $t$，波线上两点 $x_1$、$x_2$ 的相位差为：

$$\Delta\varphi = \frac{2\pi}{\lambda}(x_2 - x_1)$$

将 $\lambda = \dfrac{2\pi}{C}$ 和 $x_2 - x_1 = d$ 代入，得：

$$\Delta\varphi = \frac{2\pi}{2\pi/C} \cdot d = Cd$$

### 6.3.10 由波形曲线求波函数

**【题目】** 如题6.3.10图所示，已知 $t=0$ 时和 $t=0.5\text{s}$ 时的波形曲线分别为图中曲线(a)和(b)所示，周期 $T>0.5\text{s}$，波沿 $x$ 轴正方向传播，试根据图中给出的条件求：

1. 波函数；
2. $P$ 点的振动方程．

**(注：本题需结合题图，由波形曲线读取数据)**

**【解析】**

**(1)** 由波形图可知：

- **振幅**：$A = 0.1\ \text{m}$
- **波长**：$\lambda = 4\ \text{m}$

$t=0$ 时，原点 $O$ 处质点的位移 $y_0 = 0$，且向 $y$ 轴负方向运动（$v_0 < 0$），故初相位：

$$\varphi_0 = \frac{\pi}{2}$$

由图可知，$t=0$ 到 $t=0.5\text{s}$ 时间内，波形沿 $x$ 轴正方向移动了 $\Delta x = 2\ \text{m}$，因此波速：

$$u = \frac{\Delta x}{\Delta t} = \frac{2}{0.5} = 4\ \text{m/s}$$

频率：

$$\nu = \frac{u}{\lambda} = \frac{2}{4} = 0.5\ \text{Hz}$$

角频率：

$$\omega = 2\pi\nu = \pi\ \text{rad/s}$$

波函数的标准形式为：$y = A\cos\left[\omega\left(t - \dfrac{x}{u}\right) + \varphi_0\right]$

代入数据：

$$y = 0.1\cos\left[\pi\left(t - \frac{x}{2}\right) + \frac{\pi}{2}\right]\ \text{m}$$

化简：$y = 0.1\cos\left(\pi t - \dfrac{\pi}{2}x + \dfrac{\pi}{2}\right)\ \text{m}$

**(2)** $P$ 点的 $x$ 坐标 $x_P = 1\ \text{m}$，代入波函数得：

$$y_P = 0.1\cos\left[\pi\left(t - \frac{1}{2}\right) + \frac{\pi}{2}\right] = 0.1\cos\left(\pi t - \frac{\pi}{2} + \frac{\pi}{2}\right) = 0.1\cos(\pi t)\ \text{m}$$

### 6.3.12 由已知点的振动求波函数

**【题目】** 如题6.3.12图所示，有一平面简谐波在空间传播，已知 $P$ 点的振动方程为 $y_P = A\cos(\omega t + \varphi_0)$．

1. 分别就图中给出的两种坐标写出波函数；
2. 写出与 $P$ 点距离为 $b$ 的 $Q$ 点的振动方程．

**(注：题6.3.12图(a)中坐标原点在 $P$ 点左侧，$P$ 点坐标为 $x=l$；图(b)中坐标原点在 $P$ 点处)**

**【解析】**

**(1)** 波沿 $x$ 轴正方向传播。

**图(a)**：坐标原点在 $P$ 点左侧，$P$ 点坐标为 $x = l$。

$P$ 点的振动：$y_P = A\cos(\omega t + \varphi_0)$

波从原点传播到 $P$ 点需要时间 $\dfrac{l}{u}$，即 $P$ 点的相位比原点落后 $\omega \cdot \dfrac{l}{u}$，故原点 $O$ 处的振动为：

$$y_O = A\cos\left[\omega\left(t + \frac{l}{u}\right) + \varphi_0\right]$$

则 $x$ 轴上任意点的振动（波函数）：

$$y = A\cos\left[\omega\left(t + \frac{l}{u} - \frac{x}{u}\right) + \varphi_0\right] = A\cos\left[\omega\left(t + \frac{l-x}{u}\right) + \varphi_0\right]$$

**图(b)**：坐标原点在 $P$ 点处，$P$ 点坐标为 $x = 0$。

波沿 $x$ 轴正方向传播，$x$ 处相位比原点落后 $\omega \cdot \dfrac{x}{u}$，故波函数：

$$y = A\cos\left[\omega\left(t - \frac{x}{u}\right) + \varphi_0\right]$$

**(2)** 求 $Q$ 点的振动方程。

**图(a)**：$Q$ 点在 $P$ 点右侧，$x_Q = l + b$

$$y_Q = A\cos\left[\omega\left(t + \frac{l-(l+b)}{u}\right) + \varphi_0\right] = A\cos\left[\omega\left(t - \frac{b}{u}\right) + \varphi_0\right]$$

**图(b)**：$Q$ 点在 $P$ 点左侧（沿波传播方向的反方向），$x_Q = -b$

$$y_Q = A\cos\left[\omega\left(t + \frac{b}{u}\right) + \varphi_0\right]$$

### 9.3.19 平行带电导体板电荷分布证明

**【题目】** 证明：对于两个无限大的平行平面带电导体板来说，

1. 相向的两面上，面电荷密度总是大小相等而符号相反；
2. 相背的两面上，面电荷密度总是大小相等而符号相同．

**【解析】**

设两导体板 $A$、$B$ 的四个平面的面电荷密度依次为 $\sigma_1$、$\sigma_2$、$\sigma_3$、$\sigma_4$。

**(1)** 作一闭合高斯面，其底面分别在 $A$、$B$ 导体板内部（导体内部电场为零），侧面与板面垂直。由高斯定理：

$$\oint_S \vec{E} \cdot d\vec{S} = \frac{\sum q}{\varepsilon_0}$$

由于导体内部电场强度为零，高斯面侧面电通量为零，只有两底面有贡献，而底面在导体内部 $E=0$，故：

$$\oint_S \vec{E} \cdot d\vec{S} = 0$$

高斯面内的净电荷为零。高斯面截过两板相对的表面，包含的电荷为 $\sigma_2 \Delta S + \sigma_3 \Delta S = 0$，因此：

$$\sigma_2 + \sigma_3 = 0$$

即 $\sigma_2 = -\sigma_3$，说明相向两面上的电荷面密度大小相等、符号相反。

**(2)** 在 $A$ 板内部任取一点 $P$，该点的电场强度为零（导体静电平衡）。四个无限大均匀带电平面在 $P$ 点产生的电场强度叠加为零。

无限大均匀带电平面产生的电场强度大小为 $\dfrac{\sigma}{2\varepsilon_0}$，方向垂直板面。取向右为正方向，则 $P$ 点的合场强为：

$$\frac{\sigma_1}{2\varepsilon_0} - \frac{\sigma_2}{2\varepsilon_0} - \frac{\sigma_3}{2\varepsilon_0} - \frac{\sigma_4}{2\varepsilon_0} = 0$$

由(1)知 $\sigma_3 = -\sigma_2$，代入上式：

$$\frac{\sigma_1}{2\varepsilon_0} - \frac{\sigma_2}{2\varepsilon_0} + \frac{\sigma_2}{2\varepsilon_0} - \frac{\sigma_4}{2\varepsilon_0} = 0$$

$$\frac{\sigma_1}{2\varepsilon_0} - \frac{\sigma_4}{2\varepsilon_0} = 0$$

得 $\sigma_1 = \sigma_4$，即相背的两面上，面电荷密度大小相等、符号相同。**证毕**。

### 9.3.22 接地金属球的感应电荷

**【题目】** 半径为 $R$ 的金属球离地面很远，并用导线与地相连，在与球心相距 $d = 3R$ 处有一点电荷 $+q$，求金属球上的感应电荷的电量．

**【解析】**

金属球接地，其电势为零（与大地等电势）。设金属球上的感应电荷为 $q'$（均匀分布在球表面）。

由电势叠加原理，球心 $O$ 处的电势等于点电荷 $+q$ 和感应电荷 $q'$ 分别产生的电势之和：

$$U_O = \frac{q'}{4\pi\varepsilon_0 R} + \frac{q}{4\pi\varepsilon_0 (3R)} = 0$$

（点电荷 $+q$ 距球心 $d = 3R$）

解得：

$$\frac{q'}{R} + \frac{q}{3R} = 0 \quad\Rightarrow\quad q' = -\frac{q}{3}$$

金属球上的感应电荷电量为 $-\dfrac{q}{3}$（负号表示与点电荷 $+q$ 异号）。

### 10.3.6 载流导线在圆心处的磁感应强度

**【题目】** 如题10.3.6图所示，$AB$、$CD$ 为长直导线，$BC$ 为圆心在 $O$ 点的一段圆弧形导线，其半径为 $R$．若通以电流 $I$，求 $O$ 点的磁感应强度．

**(注：圆弧 $BC$ 对应的圆心角为 $60^\circ$，$AB$ 延长线过 $O$ 点)**

**【解析】**

$O$ 点的磁场由 $AB$、$BC$ 圆弧、$CD$ 三部分电流产生，分别计算如下：

**(1) $AB$ 段**：$AB$ 是长直导线，其延长线通过 $O$ 点。由毕奥-萨伐尔定律，电流元在 $O$ 点的 $d\vec{B}$ 方向与 $Id\vec{l}$ 和 $\vec{r}$ 的叉积有关，当 $Id\vec{l}$ 沿 $AB$ 方向、$\vec{r}$ 与 $Id\vec{l}$ 平行时，$d\vec{B} = 0$。故：

$$\vec{B}_1 = 0$$

**(2) $BC$ 段**：$BC$ 为半径为 $R$、圆心角 $\theta = 60^\circ = \dfrac{\pi}{3}$ 的圆弧导线。圆弧电流在圆心处的磁感应强度大小为：

$$B = \frac{\mu_0 I}{4\pi R}\theta$$

代入 $\theta = \dfrac{\pi}{3}$：

$$B_2 = \frac{\mu_0 I}{4\pi R} \cdot \frac{\pi}{3} = \frac{\mu_0 I}{12R}$$

方向由右手螺旋定则判定为**垂直纸面向里**。

**(3) $CD$ 段**：$CD$ 为有限长直导线。$C$ 点处 $CO$ 与 $CD$ 的夹角为 $90^\circ$，$D$ 点处 $DO$ 与 $CD$ 的夹角为 $60^\circ$。$O$ 到 $CD$ 的垂直距离 $d = R\cos60^\circ = \dfrac{R}{2}$。

有限长直导线外一点的磁感应强度公式：

$$B = \frac{\mu_0 I}{4\pi d}(\sin\theta_2 - \sin\theta_1)$$

代入 $\theta_1 = 0^\circ$、$\theta_2 = 60^\circ$：

$$B_3 = \frac{\mu_0 I}{4\pi \cdot \frac{R}{2}}(\sin60^\circ - \sin0^\circ) = \frac{\mu_0 I}{2\pi R} \cdot \frac{\sqrt{3}}{2} = \frac{\sqrt{3}\mu_0 I}{4\pi R}$$

方向为**垂直纸面向里**。

**(4) 总磁感应强度**：

$$B_O = B_1 + B_2 + B_3 = \frac{\mu_0 I}{12R} + \frac{\sqrt{3}\mu_0 I}{4\pi R}$$

整理得：

$$\boxed{B_O = \mu_0 I\left(\frac{1}{12R} + \frac{\sqrt{3}}{4\pi R}\right)}$$

方向：**垂直纸面向里**。

### 10.3.8 铁环中心处的磁感应强度

**【题目】** 如题10.3.8图所示，两根导线沿半径方向引向铁环上的 $A$、$B$ 两点，并在很远处与电源相连．已知圆环的粗细均匀，求环中心 $O$ 的磁感应强度．

**【解析】**

$O$ 点的磁场由以下几部分电流产生：

**(1) 径向导线**：$A\infty$ 和 $B\infty$ 这两段导线的延长线均通过圆心 $O$，由毕奥-萨伐尔定律，它们在 $O$ 点产生的磁感应强度均为零：$\vec{B}_{A\infty} = 0$，$\vec{B}_{B\infty} = 0$。

**(2) 两段圆弧**：设 $ACB$ 弧对应的圆心角为 $\theta$，$ADB$ 弧对应的圆心角为 $2\pi - \theta$。设流过 $ACB$ 弧的电流为 $I_1$，流过 $ADB$ 弧的电流为 $I_2$。

由于圆环粗细均匀，电阻与弧长成正比。两段圆弧并联，两端电压相等，由欧姆定律：

$$\frac{I_1}{I_2} = \frac{R_2}{R_1} = \frac{l_2}{l_1} = \frac{2\pi - \theta}{\theta}$$

圆弧电流在圆心处的磁感应强度大小为：

$$B = \frac{\mu_0 I}{4\pi R} \times (\text{圆心角})$$

$I_1$ 流过的 $ACB$ 弧对应圆心角 $\theta$（此处注意：$I_1$ 流过的是 $ACB$ 段，其对应的圆心角为 $\theta$）：

$$B_1 = \frac{\mu_0 I_1 \theta}{4\pi R}$$

方向：**垂直纸面向外**

$I_2$ 流过的 $ADB$ 弧对应圆心角 $2\pi - \theta$：

$$B_2 = \frac{\mu_0 I_2 (2\pi - \theta)}{4\pi R}$$

方向：**垂直纸面向里**

**(3) 比较 $B_1$ 与 $B_2$ 的大小**：

$$\frac{B_1}{B_2} = \frac{I_1\theta}{I_2(2\pi - \theta)} = \frac{2\pi - \theta}{\theta} \cdot \frac{\theta}{2\pi - \theta} = 1$$

所以 $B_1 = B_2$，两者大小相等。

**(4) 总磁感应强度**：

由于 $B_1$ 方向向外、$B_2$ 方向向里，大小相等：

$$\vec{B}_O = \vec{B}_1 + \vec{B}_2 = 0$$

$$\boxed{B_O = 0}$$

**结论**：环中心 $O$ 点的磁感应强度为零。

### 10.3.12 铜导线内部截面的磁通量

**【题目】** 一根很长的铜导线载有电流 $I = 10\ \text{A}$，设电流均匀分布．在导线内部作一平面 $S$，如题10.3.12图所示．试计算通过平面 $S$ 的磁通量（沿导线长度方向取长为 $1\ \text{m}$ 的一段作计算）．铜的磁导率 $\mu = \mu_0$．

**【解析】**

**(1) 求导线内部的磁感应强度分布**

设导线半径为 $R$。由安培环路定理，取半径为 $r$（$r < R$）的圆形闭合环路：

$$\oint_L \vec{B} \cdot d\vec{l} = \mu_0 \sum I_{\text{内}}$$

环路内包围的电流为：

$$\sum I_{\text{内}} = I \cdot \frac{\pi r^2}{\pi R^2} = I\frac{r^2}{R^2}$$

环路积分为 $B \cdot 2\pi r$，因此：

$$B \cdot 2\pi r = \mu_0 I\frac{r^2}{R^2}$$

解得：

$$B = \frac{\mu_0 I r}{2\pi R^2}$$

**(2) 计算磁通量**

在导线内部作一矩形平面 $S$（沿轴线方向长 $1\ \text{m}$，径向从 $0$ 到 $R$），取面积元 $dS = 1 \times dr = dr$（沿轴线方向取单位长度 $1\ \text{m}$）。

通过该面积的磁通量为：

$$\Phi_m = \int_S \vec{B} \cdot d\vec{S} = \int_0^R B \cdot dr = \int_0^R \frac{\mu_0 I r}{2\pi R^2}\,dr$$

$$= \frac{\mu_0 I}{2\pi R^2} \int_0^R r\,dr = \frac{\mu_0 I}{2\pi R^2} \cdot \frac{R^2}{2} = \frac{\mu_0 I}{4\pi}$$

代入数据：$I = 10\ \text{A}$，$\mu_0 = 4\pi \times 10^{-7}\ \text{H/m}$

$$\Phi_m = \frac{4\pi \times 10^{-7} \times 10}{4\pi} = 1.0 \times 10^{-6}\ \text{Wb}$$

$$\boxed{\Phi_m = 1.0 \times 10^{-6}\ \text{Wb}}$$

### 11.3.6 矩形线圈在通电导线旁的感应电动势

**【题目】** 如题11.3.6图所示，长直导线中通有电流 $I = 5\ \text{A}$，在其右方放一矩形线圈，两者共面．线圈长 $b = 0.06\ \text{m}$，宽 $a = 0.04\ \text{m}$，线圈以速度 $v = 0.03\ \text{m/s}$ 垂直于导线平移远离．求 $d = 0.05\ \text{m}$ 时线圈中感应电动势的大小和方向．

**【解析】**

长直导线产生的磁场为非均匀磁场，距导线 $r$ 处的磁感应强度大小为：

$$B = \frac{\mu_0 I}{2\pi r}$$

方向垂直纸面向里（由右手定则确定）。

线圈在磁场中运动，$AB$ 和 $CD$ 边的运动方向与磁场方向平行，不切割磁感线，不产生感应电动势。只有 $DA$ 边和 $BC$ 边切割磁感线产生动生电动势。

**(1) $DA$ 边**：距直导线距离为 $d$，该处的磁感应强度 $B_{DA} = \dfrac{\mu_0 I}{2\pi d}$。$DA$ 边上各点 $v$、$B$、$l$ 三者互相垂直，产生的动生电动势为：

$$\varepsilon_1 = \int_D^A (\vec{v} \times \vec{B}) \cdot d\vec{l} = B_{DA} \cdot v \cdot b = \frac{\mu_0 I}{2\pi d} \cdot v \cdot b$$

方向由 $D$ 指向 $A$（右手定则）。

**(2) $BC$ 边**：距直导线距离为 $d + a$，该处的磁感应强度 $B_{BC} = \dfrac{\mu_0 I}{2\pi(d + a)}$。产生的动生电动势为：

$$\varepsilon_2 = \int_B^C (\vec{v} \times \vec{B}) \cdot d\vec{l} = B_{BC} \cdot v \cdot b = \frac{\mu_0 I}{2\pi(d + a)} \cdot v \cdot b$$

方向由 $B$ 指向 $C$（右手定则）。

**(3) 总感应电动势**：$\varepsilon_1$ 和 $\varepsilon_2$ 在线圈回路中的方向相反，故总电动势为：

$$\varepsilon = \varepsilon_1 - \varepsilon_2 = \frac{\mu_0 I v b}{2\pi} \left(\frac{1}{d} - \frac{1}{d + a}\right)$$

代入数据：

$\mu_0 = 4\pi \times 10^{-7}\ \text{H/m}$，$I = 5\ \text{A}$，$v = 0.03\ \text{m/s}$，$b = 0.06\ \text{m}$，$d = 0.05\ \text{m}$，$a = 0.04\ \text{m}$

$$\varepsilon = \frac{4\pi \times 10^{-7} \times 5 \times 0.03 \times 0.06}{2\pi} \left(\frac{1}{0.05} - \frac{1}{0.09}\right)$$

$$= 2 \times 10^{-7} \times 5 \times 0.03 \times 0.06 \times (20 - 11.11)$$

$$= 2 \times 10^{-7} \times 0.009 \times 8.89 = 1.6 \times 10^{-8}\ \text{V}$$

$$\boxed{\varepsilon = 1.6 \times 10^{-8}\ \text{V}}$$

**方向**：由于 $\varepsilon_1 > \varepsilon_2$，总电动势方向与 $\varepsilon_1$ 一致，在线圈中为**顺时针方向**。

### 11.3.9 转动导线的动生电动势

**【题目】** 导线 $ab$ 长为 $l$，绕通过 $O$ 点的垂直轴以角速度 $\omega$ 转动，$aO = \dfrac{l}{3}$，磁感应强度 $\vec{B}$ 平行于转轴，如题11.3.9图所示．试求：

1. $a$、$b$ 两端的电势差；
2. $a$、$b$ 两端哪一点电势高？

**【解析】**

**(1) 计算动生电动势**

导线在磁场中转动，各点的线速度 $v = \omega r$，方向垂直于半径。动生电动势为：

$$d\varepsilon = (\vec{v} \times \vec{B}) \cdot d\vec{l} = B\omega r\,dr$$

**$Ob$ 段**（$r$ 从 $0$ 到 $\dfrac{2l}{3}$）：

$$\varepsilon_{Ob} = \int_0^{\frac{2l}{3}} B\omega r\,dr = \frac{1}{2}B\omega \left(\frac{2l}{3}\right)^2 = \frac{2}{9}B\omega l^2$$

方向：$O \to b$

**$Oa$ 段**（$r$ 从 $0$ 到 $\dfrac{l}{3}$）：

$$\varepsilon_{Oa} = \int_0^{\frac{l}{3}} B\omega r\,dr = \frac{1}{2}B\omega \left(\frac{l}{3}\right)^2 = \frac{1}{18}B\omega l^2$$

方向：$O \to a$

**$ab$ 间总电动势**：

$$\varepsilon_{ab} = \varepsilon_{Ob} - \varepsilon_{Oa} = \frac{2}{9}B\omega l^2 - \frac{1}{18}B\omega l^2 = \frac{4}{18}B\omega l^2 - \frac{1}{18}B\omega l^2 = \frac{3}{18}B\omega l^2 = \frac{1}{6}B\omega l^2$$

$$\boxed{\varepsilon_{ab} = \frac{1}{6}B\omega l^2}$$

**(2) 电势高低判断**

$\varepsilon_{ab} > 0$，表示电动势方向由 $a$ 指向 $b$（即 $a$ 为负极，$b$ 为正极），因此 $b$ 点电势高于 $a$ 点。

$$U_b - U_a = \varepsilon_{ab} = \frac{1}{6}B\omega l^2$$

即 **$b$ 端电势高**。

<a id="problem-11-3-10"></a>
### 11.3.10 金属杆在两平行导线间的动生电动势

**【题目】** 如题11.3.10图所示，长度为 $2b$ 的金属杆位于两无限长直导线所在平面的正中间，并以速度 $\vec{v}$ 平行于两直导线运动．两直导线通以大小相等、方向相反的电流 $I$，两导线相距 $2a$．试求金属杆两端的电势差及其方向．

**【解析】**

**(1) 磁感应强度分布**

取坐标 $r$ 以左导线为原点，向右为正。左导线在 $r$ 处产生的磁场方向垂直纸面向里，右导线在 $r$ 处产生的磁场方向垂直纸面向外。两导线通有大小相等、方向相反的电流，故磁场方向相反。

距左导线 $r$ 处的磁感应强度（取垂直纸面向里为正）：

$$B = \frac{\mu_0 I}{2\pi r} - \frac{\mu_0 I}{2\pi(2a - r)}$$

金属杆位于两导线正中间，即从 $A$（距左导线 $a - b$）到 $B$（距左导线 $a + b$）。

**(2) 动生电动势**

在金属杆上取 $dr$（距左导线 $r$），该处的动生电动势为：

$$d\varepsilon = (\vec{v} \times \vec{B}) \cdot d\vec{l} = Bv\,dr$$

总电动势：

$$\varepsilon_{AB} = \int_{a-b}^{a+b} Bv\,dr = \int_{a-b}^{a+b} \left[\frac{\mu_0 I}{2\pi r} - \frac{\mu_0 I}{2\pi(2a - r)}\right] v\,dr$$

$$= \frac{\mu_0 I v}{2\pi} \int_{a-b}^{a+b} \left(\frac{1}{r} + \frac{1}{2a - r}\right) dr$$

（注意第二项：$\displaystyle\int -\frac{dr}{2a-r} = \int \frac{d(2a-r)}{2a-r} = \ln(2a-r)$）

$$= \frac{\mu_0 I v}{2\pi} \left[\ln r - \ln(2a - r)\right]_{a-b}^{a+b}$$

$$= \frac{\mu_0 I v}{2\pi} \left[\ln\frac{r}{2a-r}\right]_{a-b}^{a+b}$$

$$= \frac{\mu_0 I v}{2\pi} \left[\ln\frac{a+b}{2a-(a+b)} - \ln\frac{a-b}{2a-(a-b)}\right]$$

$$= \frac{\mu_0 I v}{2\pi} \left[\ln\frac{a+b}{a-b} - \ln\frac{a-b}{a+b}\right]$$

$$= \frac{\mu_0 I v}{2\pi} \left[\ln\frac{a+b}{a-b} + \ln\frac{a+b}{a-b}\right]$$

$$= \frac{\mu_0 I v}{\pi} \ln\frac{a+b}{a-b}$$

$$\boxed{\varepsilon_{AB} = \frac{\mu_0 I v}{\pi} \ln\frac{a+b}{a-b}}$$

**(3) 电势高低**

令 $r_1 = a-b$，$r_2 = a+b$，则 $r_1 < r_2$。

$\dfrac{r_1}{2a-r_1} = \dfrac{a-b}{a+b} < 1$，$\dfrac{r_2}{2a-r_2} = \dfrac{a+b}{a-b} > 1$

所以 $[\ln\frac{r}{2a-r}]_{a-b}^{a+b} = \ln\frac{a+b}{a-b} - \ln\frac{a-b}{a+b} > 0$

故 $\varepsilon_{AB} > 0$，电动势方向从 $A$ 到 $B$，即 **$B$ 端电势高于 $A$ 端**。

### 12.3.5 杨氏双缝干涉

**【题目】** 在杨氏双缝干涉实验中，两条缝的间距 $d = 0.20\ \text{mm}$，缝、屏间距 $D = 1.0\ \text{m}$．

1. 若第2级明纹到屏幕中心的距离为 $6.0\ \text{mm}$，计算此单色光的波长；
2. 求相邻两明纹间的距离．

**【解析】**

**(1)** 杨氏双缝干涉明纹条件：

第 $k$ 级明纹到屏幕中心的距离为：

$$x_{\text{明}} = k\frac{D}{d}\lambda \quad (k = 0, 1, 2, \dots)$$

已知 $k = 2$ 时，$x = 6.0\ \text{mm} = 6.0 \times 10^{-3}\ \text{m}$，代入公式：

$$6.0 \times 10^{-3} = 2 \times \frac{1.0}{0.20 \times 10^{-3}}\lambda$$

解得：

$$\lambda = \frac{6.0 \times 10^{-3} \times 0.20 \times 10^{-3}}{2 \times 1.0} = 6.0 \times 10^{-7}\ \text{m} = 600\ \text{nm}$$

$$\boxed{\lambda = 600\ \text{nm}}$$

**(2)** 相邻两明纹间的距离（条纹间距）：

$$\Delta x = \frac{D}{d}\lambda = \frac{1.0}{0.20 \times 10^{-3}} \times 6.0 \times 10^{-7} = 3.0 \times 10^{-3}\ \text{m} = 3.0\ \text{mm}$$

$$\boxed{\Delta x = 3.0\ \text{mm}}$$

### 12.3.7 劳埃德镜干涉

**【题目】** 劳埃德镜干涉装置如题12.3.7图所示，镜长为 $30\ \text{cm}$，狭缝光源 $S$ 在离镜左边 $20\ \text{cm}$ 的平面内，与镜面的垂直距离为 $2.0\ \text{mm}$，光源的波长 $\lambda = 7.2 \times 10^{-7}\ \text{m}$，试求位于镜右边缘的屏幕上第1条明纹到镜的右边缘的距离．

**【解析】**

劳埃德镜干涉中，反射光有半波损失（光程突变 $\dfrac{\lambda}{2}$），反射光可视为虚光源 $S'$ 发出。

光源 $S$ 与其虚像 $S'$ 构成一对相干光源，相当于双缝干涉。两光源间距：

$$d = 2 \times 2.0\ \text{mm} = 4.0\ \text{mm} = 4.0 \times 10^{-3}\ \text{m}$$

光源到屏幕的距离 $D$ 等于镜长加上光源到镜左边缘的水平距离：

$$D = 30\ \text{cm} + 20\ \text{cm} = 50\ \text{cm} = 0.5\ \text{m}$$

由于反射光存在半波损失，两束光到达屏幕上距镜边缘 $x$ 处的光程差为：

$$\Delta = \frac{d}{D}x + \frac{\lambda}{2}$$

第1级明纹满足 $\Delta = \lambda$，即：

$$\frac{d}{D}x + \frac{\lambda}{2} = \lambda$$

$$\frac{d}{D}x = \frac{\lambda}{2}$$

解得：

$$x = \frac{\lambda D}{2d} = \frac{7.2 \times 10^{-7} \times 0.5}{2 \times 4.0 \times 10^{-3}} = \frac{3.6 \times 10^{-7}}{8.0 \times 10^{-3}} = 4.5 \times 10^{-5}\ \text{m}$$

$$\boxed{x = 4.5 \times 10^{-5}\ \text{m} = 0.045\ \text{mm}}$$

### 12.3.10 增透膜厚度

**【题目】** 在折射率 $n_1 = 1.52$ 的镜头表面涂有一层折射率 $n_2 = 1.38$ 的 $\text{MgF}_2$ 增透膜，如果此膜适用于波长 $\lambda = 550\ \text{nm}$ 的光，则膜的厚度应取何值？

**【解析】**

增透膜的原理是利用薄膜干涉使反射光干涉相消（透射光增强）。

光垂直入射，膜上表面反射光有半波损失（光从空气 $n \approx 1$ 进入 $\text{MgF}_2$ 的 $n_2 = 1.38$，$n_2 > n_{\text{空气}}$），下表面反射光也有半波损失（光从 $\text{MgF}_2$ 的 $n_2 = 1.38$ 进入镜头玻璃的 $n_1 = 1.52$，$n_1 > n_2$），两次半波损失抵消，两反射光的光程差为：

$$\Delta = 2n_2 e$$

反射光干涉相消条件：

$$\Delta = 2n_2 e = (2k + 1)\frac{\lambda}{2} \quad (k = 0, 1, 2, \dots)$$

解得膜厚：

$$e = \frac{(2k + 1)\lambda}{4n_2}$$

取 $k = 0$（最薄厚度）：

$$e = \frac{\lambda}{4n_2} = \frac{550}{4 \times 1.38} = \frac{550}{5.52} \approx 99.6\ \text{nm}$$

$$\boxed{e_{\min} = 99.6\ \text{nm}}$$

当 $k$ 取其他整数值时，膜的厚度也可取为：

$$e = 99.6 + k \cdot \frac{\lambda}{2n_2} = 99.6 + k \cdot 199.3\ \text{nm} \quad (k = 1, 2, 3, \dots)$$

在工程应用中，通常取最薄的厚度 $99.6\ \text{nm}$ 以节省材料和工艺成本。
